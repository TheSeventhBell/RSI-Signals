# RSI Trading Signals (Beginner Quant Project)
Computes RSI(14) on AAPL, generates BUY/SELL signals.
- Libraries: yfinance, Pandas, NumPy, Matplotlib
- Output: 21 BUY / 52 SELL over 1y
- Run: Open rsi_signals.ipynb in Colab
